# 🧠 Soul Scribe – AI Health & Fitness Dashboard

Soul Scribe is a modern and responsive web-based dashboard designed to track health, mood, and daily activities. It combines data visualization with an AI-inspired journaling interface to help users monitor and improve their lifestyle.

---

## 🚀 Features

* 📊 Interactive health dashboard
* 💧 Hydration tracking
* 😊 Mood tracking with activity insights
* 💤 Sleep monitoring system
* 📈 Weekly activity charts
* 🤖 AI journal-inspired interface
* 📅 Daily schedule and event planner
* 🎧 Wellness content section

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Scripting:** JavaScript (Vanilla JS)
* **Charts & Visualization:** Chart.js

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/soul-scribe.git
```

2. Navigate into the project folder

```
cd soul-scribe
```

3. Run the project

* Open `index.html` in your browser
